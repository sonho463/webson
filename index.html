スキル

スキルのイントロダクション

  • HTML

    HTMLのルールにしたがって、サイトのメッセージを適切に構造化します。そうすることで、ユーザーが検索したときに上位に表示されるようにします。

  • CSS

    サイトのページを増やしたり、ちがう製作者に頼んだりしたいと思ったときに、困らないように設計します。

  • JavaScript

    ウェブサイトにいろいろな動きを加えます。ユーザーの視線を誘導したり、知らせたい情報を強調したりします。

  • Wordpress

    ブログを書くような形でコンテンツを更新できるようにします。既存のデザインから選んでサクッと構築することもできますし、オリジナルデザインをWordPressにして運用することもできます。

  • GatsbyJS

    JAMstackという技術を使ったサイトを構築します。WordPress同様、ブログのようにコンテンツを更新できるサイトの制作もできます。サイトの表示がとても早いので、ユーザーの離脱率が下がったり、検索順位でも有利になったりします。

  • ホワイトボード・ミーティング®

    株式会社ひとまちが運営する民間の会議スキル。認定講師の資格を持っています。会議において的確で効率的な情報共有のスキルです。参考リンクは こちら

    パソコンの勉強に使えるリンク集

    学校で、プログラミング教育が導入されるということで、苦手な先生のクラスでも、プログラミングのサイトにアクセスできるようにしたいという課題がありました。所属しているコミュニティ内の勉強会で作ったものです。

    パソコンのべんきょうのためのリンク

    １つ目の「パソコンの勉強に使えるリンク集」のあと、もう少しこんなのにしたい！という思いをかなえるために作り上げました。これもコミュニティの勉強会で作りました。

    ぼちぼちブログ

    WordPressブログ。無料テーマCocoonを使用しています。自分の経験を発信できればと思って作りました。広告やアフィリエイトなども少しここで体験しようと思っています。

    八尾市教職員組合　新規採用者向けLP

    ライティング、情報設計、デザイン、コーディングをしました。ターゲットが市内の教職員なので、基本的な流入経路はチラシにつけたQRコードです。

    Gatsby-microCMS連携ブログサイト

    フロントにGatbyjs ヘッドレスCMSにmicroCMSを使ってブログサイトを作成。デザインはstartBootstrapのテンプレートを使用しました。
